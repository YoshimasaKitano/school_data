type()→調べるときに使う関数

int型は整数値を表すデータ型

float型は浮動小数点型

str型は文字列型

変数の中に文字列を入れると文字列型になる(str型)
(例)
name = "竹國聡子"
print(type(name)) ### <class 'str'>

変数の中に数字を入れるとint型になる
(例)
name = 300
print(type(name)) ### <class 'int'>

変数の中にいろいろ入れてしまって型が変わってしまうトラブルが起こりやすいので注意

割り算で計算するとfloat型になる
(例)
name = name / 5
print(type(name)) ### <class 'float'>

エラー回避するためにあらかじめ型を指定して無理やり実行に移すことはできる→バグではある

str()→文字列に変換する関数

str()は整数値でも浮動小数点でもstr型に変換することができる
(例)
num = 10
print(type(num)) ### <class 'int'>

str_num = str(num)
print(str_num) ### 10
print(type(str_num)) ### <class 'str'>

fnum = 3.14
str_fnum = str(fnum)
print(fnum) ### 3.14
print(type(str_fnum)) ### <class 'str'>

int()→整数値型に変換する

文字列や浮動小数点が入った変数はint型に変換できない→浮動小数点型に変換してからint型に変換することは可能

float()→浮動小数点型に変換

float型をint型に変換すると小数点以下が切り捨てられる
(例)
f_fnum = float(str_fnum)
print(f_fnum) ### 3.14
print(type(f_fnum)) ### <class 'float'>

int_fnum = int(f_fnum)
print(type(int_fnum)) ### <class 'int'>
print(int_fnum) ### 3 小数点以下が切り捨てられる

float型をint型に変換したものをもう一度float型に変換しても切り捨てられた小数点以下は戻ってこないので注意
(例)
f_fnum = float(int_fnum)
print(type(f_fnum)) ### <class 'float'>
print(f_fnum) ### 3.0

変数名を安易につけない→正しい変数名をつける

bin()→2進数に変換

oct()→8進数に変換

hex()→16進数に変換

printで表示するときは10進数になる

Pythonには定数がない

ただし大文字で書かれているものは定数なので、この値を上書きしたりするのはNG→Pythonの作法

変数のつけ方はたくさんあるがアンダースコアを使う付け方(スネークケース)がPyhonでは一般的
(例)
myName→キャメルケース
my_name→スネークケース

my-name→ケバブケース(HTML,CSSでよく使う変数のつけ方でPythonでは使えない)

仕事では変数名をコーディング規約に従ってキャメルケースかスネークケースに統一する

Pythonの予約語(もともと関数の名前だったりするもの)を変数に使うことはできない

大文字と小文字でそれぞれ別の変数になることに注意

演算子→+や=などの記号のこと

複合代入演算子→+=など
a += 1 は、a = a + 1と同じ処理
a -= 1 は、a = a - 1と同じ処理
a *= 2 は、a = a * 2と同じ処理
a /= 2 は、a = a / 2と同じ処理
a %= 2 は、a = a % 2と同じ処理
a **= 3 は、a = a ** 3と同じ処理

文字列では+=、*=が使える
(例)
who = "猫"
text = ""

text = "我が輩は"
print(text) ### 我が輩は
text += who #text = text + who
print(text) ### 我が輩は猫

text += "である。"
print(text) ### 我が輩は猫である。

**=は文字列には使えないので注意

変数の中身を移し替えてもそのまま中身は残ったままになる→その変数に新しい値を代入したら値が変わる
(例)
a = "A"
b = "B"

# 交換処理
t = a
print(a) ### A
a = b
print(a) ### B
b = t

print("変数a:",a) ### 変数a: B
print("変数b:",b) ### 変数b: A

リテラル→コードに直接書いた数字や文字列のこと。実数が入っている。

リファレンス→実際に値は入らずそこのアドレスが入ってること。

組み込み関数→いつでもどこでも呼び出せる関数

引数は１つしか入れることができない関数もあれば、複数の引数を入れることができる関数もある。

abs()→絶対値を返す関数

max()→引数の最大値を返す

min()→引数の最小値を返す

pow(x,y)→xのy乗を返す

pow(x,y,z)→xのy乗をzで割ったあまりを返す

round(x,y)→xを小数点第y位で丸め処理を行う

アルファベット1文字は1byte、日本語1文字は2byte

len()→引数の文字列の文字数を返す

chr()→引数のUnicodeを文字列で返す

Unicode→世界中の文字をコンピュータ上で一貫して扱うための国際的な標準規格のこと。

ord()→引数の文字をUnicodeで返す
文字列ではなく文字のUnicodeを返す→引数に文字列を入れるとエラーになってしまう

input()→キーボードからの入力を取得する関数
inputの入力を止めるときはCtrl+C

☆inputで入力されたデータはすべて文字列型になる。→数字を入れても文字列型になるので計算などに注意
