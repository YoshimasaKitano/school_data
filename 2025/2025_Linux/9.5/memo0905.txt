パッケージ管理
　コマンドやアプリケーションの管理
　RHEL系：dnfコマンド(古いコマンドyum)
　Debian系：aptコマンド

　★パッケージのインストール
　　構文：dnf install パッケージ名

　　オプション：-y インストール途中に yes/noの確認を自動的に進めてくれる。

　　例：dnf install -y tar

　★パッケージのインストール確認
　　構文：dnf info パッケージ名

　　例：dnf info tar

　★インストール済みパッケージの一覧取得
　　構文： dnf list installed

　　※部分的にしかパッケージの名称がわからないときパイプでgrepで絞り込みをかけて探す。

　　例：dnf list installed | grep zip

　★パッケージの削除
　　構文：dnf remove パッケージ名

　　例：dnf remove tree

アーカイブと圧縮

　アーカイブ：複数のファイル/ディレクトリを1つのアーカイブファイルにまとめる
	　　　コマンド：tar

　圧縮：圧縮アルゴリズムを使ってファイルの容量を小さくする
	コマンド：gzip, bzip2, zip/unzip

　★tarコマンド

　・アーカイブファイルの生成
	tar cvf アーカイブファイル名 ターゲットファイル名/ディレクトリ名

　・アーカイブファイルの中身を確認
	tar tvf アーカイブファイル名

　・アーカイブファイルを展開する
	tar xvf アーカイブファイル名

　vオプション：tarコマンド実行時に、ファイルやディレクトリ名を表示します。
	　　　 tと組み合わせたときは、パーミッションも表示
	　　　※基本つけたほうがわかりやすい

　zオプション：tarコマンドで、tar.gz形式で、アーカイブ圧縮/解凍アーカイブ展開を行う

	　　　※アーカイブと圧縮を同時に実行　czf
		解凍アーカイブ展開を同時に実行 xzf

　★gzipコマンド

　・ファイルを圧縮する
	gzip ターゲットファイル名

　・圧縮ファイルを展開する
	gzip -d 圧縮ファイル名　または、
	gunzip 圧縮ファイル名

　・圧縮ファイルを別名で保存する
	gzip -c ターゲットファイル名 > 圧縮ファイル別名.gz

　　-cオプションは、圧縮ファイルの内容(バイナリ)を標準出力(画面出力)する。
　　これをファイルリダイレクトすることで別名で保存できる。

　★bzip2コマンド(gzipコマンドより圧縮率が高い)

　・ファイルを圧縮する
	bzip2 ターゲットファイル名

　・圧縮ファイルを展開する
	bzip2 -d 圧縮ファイル名　または、
	bunzip2 圧縮ファイル名

　・圧縮ファイルを別名で保存する
	bzip2 -c ターゲットファイル名 > 圧縮ファイル別名.gz

　　-cオプションは、圧縮ファイルの内容(バイナリ)を標準出力(画面出力)する。
　　これをファイルリダイレクトすることで別名で保存できる。
